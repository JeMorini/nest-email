## Description

Microservice with Nest and SendGrid for sending emails </br>
Tutorial for this project on Medium: https://medium.com/@jemorini13/nest-js-sendgrid-criando-micro-servi%C3%A7o-de-envio-de-email-e8a74715d1e8

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod

```
